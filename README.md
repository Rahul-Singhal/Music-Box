    <article class="markdown-body entry-content" itemprop="mainContentOfPage"><h1>
<a name="music-box" class="anchor" href="#music-box"><span class="octicon octicon-link"></span></a>Music-Box</h1>

<p>An animated music box created in openGL.</p>

<h3>
<a name="contents" class="anchor" href="#contents"><span class="octicon octicon-link"></span></a>CONTENTS:</h3>

<hr><p>This tar archive contains following folders with listed files:</p>

<pre><code>1. src: Contained files are
    1.textureLoader.h
    2.1.textureLoader.cpp
    3.man.h
    4.man.cpp
    5.box.h
    6.box.cpp
    7.main.cpp

2. textures: Contained files are
    1.cap1.bmp
    2.shirt21.bmp
    3.shirt22.bmp
    4.skin1.bmp
    5.skin2.bmp
    6.skin3.bmp
    7.wood1.bmp
    8.wood2.bmp
    9.wood3.bmp

3. Makefile: To copile and generate the binaries.

4. README: This file.
</code></pre>

<h3>
<a name="how-to-run" class="anchor" href="#how-to-run"><span class="octicon octicon-link"></span></a>HOW TO RUN:</h3>

<hr><ol>
<li>Extract the archive and change the directory into that folder.</li>
<li>Run, "make". This will generate the executable binary in the bin folder.</li>
<li>Run that binary using the command "./bin/music_box"</li>
</ol><h3>
<a name="instructions-to-interact-with-the-objects" class="anchor" href="#instructions-to-interact-with-the-objects"><span class="octicon octicon-link"></span></a>INSTRUCTIONS TO INTERACT WITH THE OBJECTS:</h3>

<hr><p>There are 4 modes of interaction that we provide in this package.</p>

<h4>
<a name="mode-1" class="anchor" href="#mode-1"><span class="octicon octicon-link"></span></a>MODE 1:</h4>

<hr><ol>
<li>You can rotate the box in any direction by pressing the left mouse button and dragging it to apply the desired rotation.</li>
<li>Click, drag and enjoy!</li>
</ol><h4>
<a name="mode-2" class="anchor" href="#mode-2"><span class="octicon octicon-link"></span></a>MODE 2:</h4>

<hr><ol>
<li>In this mode we enable the user to interact with the camera capturing the entire world. </li>
<li>To activate this mode press the keys 'Z' and 'N' in order.</li>
<li>Use 'UP' and 'DOWN' arrow keys to move the camera towards and away from the objects respectively.</li>
<li>Use 'LEFT' and 'RIGHT' arrow keys to move the camera to the left and right respectively.</li>
<li>Use 'PAGE_UP' and 'PAGE_DOWN' keys to rotate the camera clockwise and anti-clockwise about its current position</li>
</ol><h4>
<a name="mode-3" class="anchor" href="#mode-3"><span class="octicon octicon-link"></span></a>MODE 3:</h4>

<hr><ol>
<li>In this mode, the user is able to control the box using the keyboard keys.</li>
<li>Use the key 'O' to open the box.</li>
<li>Use the key 'C' to close the box.</li>
</ol><h4>
<a name="mode-4" class="anchor" href="#mode-4"><span class="octicon octicon-link"></span></a>MODE 4:</h4>

<hr><ol>
<li>
